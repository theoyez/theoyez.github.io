FAQ

Short answers to common questions.

Why do submissions go through GitHub, not a web form?

It gives an audit trail. Every score is a PR with report.json/report.html attached; CI verifies numbers match. That keeps the board credible.

What’s the difference between DERIVED and STRICT?

DERIVED uses the published manifest + hash (easier). STRICT additionally verifies a cryptographic signature we publish; reports include signature_verified: true when authentic.

Do you host my data or model?

No. You run locally and submit only the reports + one CSV line.

I’m new to GitHub. How do I submit?

Follow the copy-paste steps in the Submission guide. It’s fork → run → add files → open PR.

Where can I ask questions or collaborate?

Email contact@theoyez.org.
