What is TrustEval-Mini?

A small, auditable RAG evaluation that checks faithfulness and attribution on a fixed manifest of sources and questions.

Leaderboard · GitHub · Submission guide

Provenance Modes

  • DERIVED — quickest start. Uses published manifest + hash.
  • STRICT — cryptographically verified. We sign the manifest; the evaluator verifies with our public key. Reports include signature_verified: true when authentic.

How scoring gets on the board

Run locally → produce report.json/report.html → add one row to scores.csv → open a Pull Request. Our checks confirm your row matches your report before merge.
